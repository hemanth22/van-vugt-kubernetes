To get your RHCSA, you'll need to know how to work with storage. In this challenge you have to set up a complete storage environment that is based on the loop device. 

In this challenge you will practice the following:

*       Using dd to create an image file
*       Using losetup to create a loop device
*       Using fdisk and mkfs to create a partition and filesystem on top of the loop device
