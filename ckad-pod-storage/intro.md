Kubernetes offers advanced options for working with volumes, which can be configured internally as well as through Persistent Volumes provided by the cloud. In this interactive scenario you will learn how to configure Pods to use Persistent Storage, thus allowing you to run stateful applications in Kubernetes.
 
 In the following steps you will learn the following:
 
*	Configuring Pod Volumes
*	Providing Persistent Volumes
*	Using Persistent Volume Claims to connect Pods to Persistent Volumes
